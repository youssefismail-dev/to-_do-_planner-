# to-_do-_planner-
A personal learning project built with React.
## Getting Started

### 1. Clone the repository
```bash
git clone <repository-link>
```
### 2. Install dependencies
```bash
npm install
```

### 3. Start the development server
```bash
npm start
```

The application will open at:
```
http://localhost:3000
```

## About

This project was built from scratch as a learning exercise. It recreates the functionality of a Week To-Do Planner and was developed to improve my React skills.
